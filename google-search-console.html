Configuration Google Search Console

  1. Aller sur Google Search Console
  2. Ajouter votre propriété (ostiro-network.com)
  3. Choisir la méthode de vérification "Balise HTML"
  4. Copier le code de vérification
  5. Remplacer "YOUR_GOOGLE_SEARCH_CONSOLE_VERIFICATION_CODE" dans tous les fichiers HTML
